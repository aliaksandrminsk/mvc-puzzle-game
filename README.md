# Merge Game

This project is a merge game. Technologies: [PixiJS](https://pixijs.com/), [TypeScript](https://www.typescriptlang.org/) and [Webpack](https://webpack.js.org/)

To run the package.

```
git clone https://github.com/aliaksandrminsk/mvc-puzzle-game
cd mvc-puzzle-game
npm install
npm run start
```
